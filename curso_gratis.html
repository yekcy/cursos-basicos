CREACIÓN DE PAGINAS WEB

Los sitios web son conocidos como aquel grupo de archivos html unidos a través de hipervínculos almacenados en un servidor web, lo que significa un equipo o PC

que alberga diversas páginas web lo que implica la conexión a internet constantemente. Los diseños web es una actividad la cual consiste en planificar,diseñar, e implementar y establecer mantenimientos de sitios web. Factores como la navegabilidad, la interactividad y la arquitectura de la información hacen parte del diseño web y el diseño de interfaz, todo este conjunto de conceptos e ideas logran una asequibilidad en la vinculación de medios en los que podriamos mencionar: audio, imagen, texto, enlaces y video.

ACERCA DE HTML

HTML, es uno de los lenguajes de programación más utilizados para el desarrollo de páginas web, sus siglas:HyperText Markup Language el cual traduce: Lenguaje de Formato de Documentos para Hipertexto. Dentro de las características tenemos:

  • Es un lenguaje donde ofrece opciones muy útiles para el libre desarrollo de un sitio web
  • Dentro de una página este lenguaje es ideal para anexar palabras claves y sintaxis como cuando se aplica en cualquier otro idioma habitual
  • El HTML se fundamenta por una serie de códigos escritos en un registro de texto por el desarrollador del sitio web.
  • Utiliza etiquetas para lograr una buena visión
